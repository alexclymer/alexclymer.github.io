This Website

This portfolio site itself demonstrates the skills I bring to my technical documentation work. Rather than relying on a website builder, I designed and built it from scratch as a static HTML/CSS site (with some help from Claude of course!). I applied the same docs-as-code principles I use professionally.

The site is version-controlled in Git and hosted on GitHub Pages with a custom domain configured through DNS. Design decisions were made deliberately: a shared stylesheet with CSS custom properties for easy theming and a responsive layout.

Transforming Legacy Content

In 2018, I inherited the STA documentation. Having fresh eyes, I was able to approach the doc set as a new user would. I found it confusing and difficult to locate information. Over the course of a year, I completely restructured the content.

  • Consolidated the number of guides from eight to four.
  • Reduced the overall content by ~50% (1220 pages to 625) while still retaining all user-critical information.
